# User Story: View Contribution Compliance Report

**As a** Member of a stokvel group,  
**I want to** view contribution compliance reports over time,  
**So that** I can monitor my contribution consistency and payment history.

---

## Acceptance Criteria

| ID | Acceptance Criteria |
|----|---------------------|
| AC1 | The system shall display a compliance score percentage and monthly breakdown table showing expected vs actual payments with status (On Time, Late, Missed) |
| AC2 | The system shall provide personalized recommendations based on the member's compliance rate |

---

## User Acceptance Tests

| # | Test Description |
|---|------------------|
| UAT-01 | Verify that member can view compliance score percentage and monthly breakdown table |
| UAT-02 | Verify that personalized recommendations appear based on compliance rate |
