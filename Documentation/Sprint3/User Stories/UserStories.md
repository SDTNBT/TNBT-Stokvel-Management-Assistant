# Stokvel Management System - User Stories

## User Story #1: Invite New Members (Admin) - Thabiso

**As an** Admin of a stokvel group,  
**I want to** invite new members to join the group,  
**So that** they can become part of the stokvel and start contributing.

**Acceptance Criteria:**
- Admin can add new members by entering their email address
- System sends an invitation email to the new member

---

## User Story #2: Online Contributions via Payment Gateway (Member) - Nkateko

**As a** Member of a stokvel group,  
**I want to** make contributions online via a third-party payment gateway,  
**So that** I can pay conveniently and securely without handling cash.

**Acceptance Criteria:**
- Member can make payments using Stripe payment gateway
- Payment status (Confirmed/Pending) is recorded in the system

---

## User Story #3: Save Bank Account Details (User) - Sikhona

**As a** User of the stokvel system,  
**I want to** securely save my bank account details in my profile,  
**So that** the system has a record of where to send my payouts.

**Acceptance Criteria:**
- User can add, edit, and delete bank account details
- Bank details are stored securely (encrypted)

---

## User Story #4: Manage Payout Schedule (Treasurer) - Gomolemo

**As a** Treasurer of a stokvel group,  
**I want to** manage the payout schedule,  
**So that** funds are distributed correctly and on time.

**Acceptance Criteria:**
- Treasurer can create and edit payout schedules
- Members receive notifications before payout date

---

## User Story #5: View Prime Lending Rate & Projected Savings (Member) - Liso

**As a** Member of a stokvel group,  
**I want to** view the current South African prime lending rate and see my projected savings growth,  
**So that** I can estimate future returns.

**Acceptance Criteria:**
- Member dashboard displays current South African prime lending rate
- System calculates projected savings growth based on current rates

---

## User Story #6: View Contribution History (Member) - Kagiso

**As a** Member of a stokvel group,  
**I want to** view my complete contribution history,  
**So that** I can track my payments over time.

**Acceptance Criteria:**
- Member can view table showing date, group name, amount, and status
- Member can filter contributions by group name and status
